# Use Cases

The application is descried in use cases. The details of the use cases shall
be described in files grouped in subfolders per category.

## User

* UC-US-001 - user registration
* UC-US-002 - password reset
* UC-US-003 - login
* UC-US-004 - send group invitation

## Group

* UC-GR-001 - create new group
* UC-GR-002 - manage trainers
* UC-GR-003 - manage players
* UC-GR-004 - manage guardians
* UC-GR-005 - edit guardian
* UC-GR-006 - manage shirt sets

## Event

* UC-EV-001 - create event
* UC-EV-002 - invite players
* UC-EV-003 - select players for teams
* UC-EV-004 - change invitation status
* UC-EV-005 - assign shirt set

## Statistics

* UC-ST-001 - Player Statistics
* UC-ST-002 - Event Attendance Overview
* UC-ST-003 - Team Selections
